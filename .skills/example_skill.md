# Example Skill

This is an example skill file that demonstrates the DevMate skill format.

## Frontmatter Fields

- **name**: Unique identifier for the skill
- **description**: What this skill does
- **trigger_keywords**: Keywords that trigger this skill

## Usage

When the agent detects trigger keywords in a user query, it will reference this skill's content to provide more contextual and accurate responses.

## Best Practices

1. Keep skills focused on a single topic
2. Include code examples when relevant
3. Document edge cases and common pitfalls
4. Update skills as the project evolves
